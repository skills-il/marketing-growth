
# סדרות אימייל ישראליות

## הוראות

### שלב 1: ציות לחוק הספאם (תיקון 40)

**דרישות חוקיות (תיקון 40 לחוק התקשורת, בזק ושידורים, 1982):**

חוק הספאם הישראלי הוא מהמחמירים בעולם. דרישות מרכזיות:

| דרישה | פרטים |
|-------|-------|
| הסכמה מוקדמת | חובה לקבל הסכמה מפורשת ומתועדת לפני שליחת דואר מסחרי |
| סוג הסכמה | נדרשת Opt-in (לא Opt-out). המשתמש חייב להסכים באופן פעיל |
| זיהוי | כל אימייל חייב לזהות בבירור את השולח (שם ופרטי קשר) |
| הסרה | חובה לכלול מנגנון הסרה תקין בכל אימייל מסחרי |
| זמן הסרה | חובה לכבד בקשות הסרה תוך 3 ימי עסקים |
| שמירת רשומות | חובה לשמור רשומות הסכמה ולהוכיח הסכמה אם נדרש |
| קנסות | עד 50,000 ש"ח לכל הפרה ללא הוכחת נזק |

**מה נחשב "דואר מסחרי" לפי החוק:**
- אימיילים של שיווק ישיר
- ניוזלטרים פרסומיים
- הודעות על מוצרים
- הודעות על מבצעים והנחות

**חריגים (לא נדרשת הסכמה):**
- לקוחות קיימים: מותר לשלוח על מוצרים דומים למה שרכשו, אם ניתנה אפשרות הסרה בעת הרכישה
- אימיילים טרנזקציוניים (אישורי הזמנה, קבלות, עדכוני משלוח)

### שלב 2: רינדור RTL באימייל בעברית

**מבנה אימייל RTL:**
```html
<html dir="rtl" lang="he">
<head>
  <meta charset="UTF-8">
  <style>
    body { direction: rtl; text-align: right; font-family: Arial, Tahoma, sans-serif; }
    table { direction: rtl; }
  </style>
</head>
```

**תמיכת RTL בתוכנות אימייל בישראל:**
| תוכנת אימייל | נתח שוק (ישראל) | תמיכת RTL | הערות |
|-------------|-----------------|-----------|-------|
| Gmail | כ-45% | טובה | מכבד dir="rtl" |
| Outlook (שולחני) | כ-20% | בינונית | דורש MSO conditional comments |
| Apple Mail | כ-15% | טובה | תמיכה מלאה |
| Yahoo Mail | כ-5% | טובה | RTL סטנדרטי |
| Outlook.com | כ-10% | בינונית | דומה ל-Outlook שולחני |

**בעיות RTL נפוצות ותיקונים:**
| בעיה | תיקון |
|------|-------|
| יישור טקסט מתהפך ב-Outlook | הוסיפו `dir="rtl"` על כל `<td>` |
| נקודות תבליט בצד הלא נכון | השתמשו ב-`unicode-bidi: embed` |
| מספרים מופיעים הפוך | עטפו מספרים ב-`<span dir="ltr">` |
| תמונות לא מיושרות | השתמשו ב-`align="right"` |

### שלב 3: אופטימיזציה של שורות נושא בעברית

**מדדי אימייל ישראליים:**
| מדד | ממוצע ישראלי | ביצוע טוב |
|-----|-------------|-----------|
| שיעור פתיחה | 20-25% | 30%+ |
| שיעור הקלקה | 2-4% | 5%+ |
| שיעור הסרה | 0.2-0.5% | פחות מ-0.2% |
| יום שליחה מיטבי | שלישי, רביעי | בדקו לפי קהל |
| שעת שליחה מיטבית | 9-10 בבוקר, 19-20 | בדקו לפי קהל |

**נוסחאות שורות נושא בעברית:**
```
1. דחיפות: "רק היום: [הצעה]"
2. אישי: "[שם], יש לנו משהו בשבילך"
3. סקרנות: "לא תאמינו מה גילינו על [נושא]"
4. תועלת: "איך [תועלת] ב-[זמן]"
5. הוכחה חברתית: "1,000+ ישראלים כבר [פעולה]"
6. שאלה: "עדיין [בעיה]?"
```

**שיטות מומלצות לשורות נושא:**
- שמרו על 40-50 תווים (עברית תופסת יותר מקום ויזואלי)
- טקסט תצוגה מקדימה (preheader) חשוב באותה מידה; 40-90 תווים
- פרסונליזציה עם שם פרטי מעלה שיעורי פתיחה 15-20%
- אימוג'ים בשורות נושא מעלים שיעורי פתיחה אבל בדקו לפי קהל

### שלב 4: סדרות אימייל לפי חגים ישראליים

**לוח חגים יהודי לקמפיינים:**
| חג | עיתוי | סוג אימייל | ערכות נושא |
|----|-------|-----------|-----------|
| ראש השנה | ספט/אוק | ברכה + מבצע | התחלות חדשות, חידוש, דבש ותפוחים |
| יום כיפור | ספט/אוק | לפני החג בלבד | סליחה, התבוננות (ללא אימייל מסחרי ביום עצמו!) |
| סוכות | ספט/אוק | מבצע, אירועים | חוץ, משפחה, אירוח |
| חנוכה | נוב/דצמ | מדריכי מתנות, מבצעים | מתנות, אור, משפחה, 8 ימי דילים |
| פורים | פבר/מרץ | קמפיינים שמחים | תחפושות, כיף, משלוחי מנות |
| פסח | מרץ/אפר | ניקיון אביב, מבצעים | חירות, ניקיון, משפחה, כשרות |
| יום הזיכרון | אפר/מאי | מכובד, ללא מכירות | יום זיכרון - אין אימייל מסחרי בהחלט! |
| יום העצמאות | אפר/מאי | פטריוטי, מבצעים | כחול-לבן, גאווה ישראלית, על האש |
| שבועות | מאי/יוני | חלבי, למידה | עוגות גבינה, ערכות לבנות |

**כללים קריטיים:**
- לעולם אל תשלחו אימייל מסחרי ביום כיפור
- לעולם אל תשלחו אימייל מסחרי ביום הזיכרון
- הפחיתו תדירות אימייל בשבת
- רגישות תרבותית: יום השואה דורש טון חגיגי או ללא אימייל

### שלב 5: סוגי סדרות אימייל לשוק הישראלי

**סדרת קבלת פנים (5 אימיילים):**
```
אימייל 1 (מיידי): ברוכים הבאים + מתן ערך
  נושא: "ברוכים הבאים ל[מותג]! 🎉"

אימייל 2 (יום 2): סיפור המותג
  נושא: "הסיפור שלנו - למה הקמנו את [מותג]"

אימייל 3 (יום 4): ערך מרכזי
  נושא: "[שם], הנה [תועלת מרכזית] שהבטחנו"

אימייל 4 (יום 7): הוכחה חברתית
  נושא: "מה אומרים [מספר] הלקוחות שלנו"

אימייל 5 (יום 10): דחיפה להמרה
  נושא: "מתנה מיוחדת רק בשבילך, [שם]"
```

**סדרת עגלה נטושה (3 אימיילים):**
```
אימייל 1 (שעה): תזכורת
  נושא: "שכחת משהו בעגלה 🛒"

אימייל 2 (24 שעות): דחיפות
  נושא: "הפריטים שלך עדיין ממתינים!"

אימייל 3 (72 שעות): תמריץ
  נושא: "הנחה מיוחדת על העגלה שלך - רק היום"
```

**סדרת שיחזור (3 אימיילים):**
```
אימייל 1 (30 יום לא פעיל): צ'ק-אין
  נושא: "התגעגענו אליך, [שם]!"

אימייל 2 (45 יום): תמריץ
  נושא: "מתנה מיוחדת כי חסרת לנו"

אימייל 3 (60 יום): הזדמנות אחרונה
  נושא: "נשארים בקשר? [שם], זו ההזדמנות האחרונה"
```

### שלב 6: בדיקות ואופטימיזציה

**A/B Testing לאימייל ישראלי:**
- בדקו שורות נושא בעברית מול עברית+אימוג'י
- בדקו רישום פורמלי מול בלתי פורמלי
- בדקו שעות שליחה: בוקר (9-10) מול ערב (19-20)
- בדקו טקסט CTA: ישיר ("קנו עכשיו") מול רך ("לפרטים נוספים")
- בדקו שם שולח: שם מותג מול שם אישי (שם מנכ"ל/מייסד עובד טוב בישראל)

**עבירות (Deliverability) לספקי אימייל ישראליים:**
- רוב האימייל הישראלי הוא Gmail, Outlook ו-Yahoo
- השתמשו באימות שליחה: SPF, DKIM, DMARC
- שמרו על רשימות נקיות
- חממו דומיינים חדשים בהדרגה

## דוגמאות

### דוגמה 1: סדרת קבלת פנים ל-SaaS ישראלי
**קלט**: "צרו סדרת אימיילי קבלת פנים ל-SaaS ישראלי לניהול פרויקטים"
**פלט**: סדרה של 5 אימיילים עם שורות נושא בעברית, תבניות HTML ב-RTL, טון ישראלי קז'ואל, ציות לחוק הספאם.

### דוגמה 2: קמפיין חגים לחנות מקוונת
**קלט**: "תכננו קמפיינים לפי לוח החגים היהודי לחנות אונליין ישראלית"
**פלט**: לוח שנה מלא ממופה לחגים, עם שורות נושא, ערכות נושא ותזמון שליחה. כולל תאריכי חסימה וכללי רגישות תרבותית.

### דוגמה 3: תבנית אימייל RTL
**קלט**: "בנו תבנית HTML לאימייל שמרונדרת נכון בעברית בכל תוכנות האימייל"
**פלט**: תבנית RTL מבוססת טבלאות עם Outlook conditional comments, מחסנית גופנים בעברית, כפתורי CTA בטוחים ל-RTL.

## פתרון בעיות

- **בעיה**: טקסט עברי מרונדר שמאל-לימין בחלק מתוכנות האימייל
  **פתרון**: הוסיפו `dir="rtl"` לכל `<table>`, `<td>` ו-`<div>`, לא רק למעטפת החיצונית. השתמשו בסגנונות inline. בדקו ב-Litmus או Email on Acid.

- **בעיה**: קישור הסרה לא תואם לחוק הספאם
  **פתרון**: ההסרה חייבת להיות בעברית ("להסרה מרשימת הדיוור"), נראית בבירור, ופונקציונלית תוך 3 ימי עסקים. כללו זיהוי שולח (שם חברה, כתובת, טלפון).

- **בעיה**: שיעורי פתיחה נמוכים ביחס למדדים
  **פתרון**: בדקו שעות שליחה מול לוח הזמנים הישראלי (ראשון יום עבודה, שישי קצר). פרסנלו שורות נושא עם שמות פרטיים בעברית. ודאו שטקסט תצוגה מקדימה מוגדר ולא מציג קוד HTML.
